To start, open the site you used in the last challenge and add some content into each div.

Then, you need to figure out what colors you want to use in your design and what other elements you may need (like borders!) to make it look good.

When you're done, push your files to the GitHub repo you set up for this project. If you need a refresher, refer to the Push Your Local Repo to GitHub step from the Git, GitHub, and the Command Line class!
